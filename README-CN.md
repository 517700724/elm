# vue-ele

# Libraries
下面是一些核心的依赖
- vue（^2.5.2）
- vue-router（^3.0.1）：vue官方的路由插件
- vue-resource：提供了web请求服务的vue插件
- express：node.js的web开发框架
- stylus：css预处理器

# Feature
下面是你可能会注意到的一些特征
- vue 2.0 + express + stylus
- 数据mock
- 处理移动端一像素
- stricky footer 布局
